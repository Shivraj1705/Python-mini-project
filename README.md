# Smart Personal Finance Manager with Expense Prediction

![Python](https://img.shields.io/badge/Python-3.9%2B-blue)
![Pandas](https://img.shields.io/badge/Pandas-2.0%2B-green)
![License](https://img.shields.io/badge/License-MIT-yellow)

A lightweight **personal finance tracker** built in Python that helps you:
- Track daily expenses by category
- Detect overspending against budgets
- Predict next month's expenses using time-series forecasting
- Visualize spending trends with interactive charts
- Get personalized saving tips

Perfect for beginners to learn **Pandas, Matplotlib, Statsmodels**, and **Streamlit** while building a real-world fintech mini-project.

---

## Features

| Feature | Description |
|-------|-----------|
| **Expense Tracking** | Add, edit, delete expenses with category, amount, date |
| **Budget Alerts** | Warns when spending exceeds 80% of category budget |
| **Expense Prediction** | Uses **Exponential Smoothing** to forecast next month's spending |
| **Visualizations** | Pie charts, bar plots, trend lines |
| **Saving Tips** | Smart suggestions based on spending patterns |
| **Web Interface** | Clean frontend using **Streamlit** |
